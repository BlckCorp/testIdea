# testIdea

Репозиторий для идей, которые мы придумываем, проверяем и превращаем в MVP.

## Зачем это репо

Здесь храним не просто случайные мысли, а заготовки под реальные продукты и услуги:

- подарки для пар;
- сайты-приглашения;
- услуги для Авито;
- идеи для малого бизнеса;
- ИИ-инструменты;
- электрика и бытовые услуги;
- документы, лендинги, упаковка объявлений.

## Как устроено

```text
ideas/       — оформленные идеи и продуктовые линейки
templates/   — шаблоны карточек, офферов и объявлений
backlog/     — сырые идеи, которые ещё надо разобрать
```

## Правило репозитория

Каждая идея должна отвечать на три вопроса:

1. Что продаём?
2. Кому это нужно?
3. Как сделать MVP за 1 вечер?

## Быстрый приоритет

Сначала развиваем идеи, которые можно быстро проверить через Авито, ВК, личные чаты или простой лендинг.
